No server. No account. No tracking. Your templates live in your browser. Verify it yourself — search the source for fetch( and you won't find a single external URL.

What you get

Four ways to paste

Keyboard shortcuts, popup menu, /slash commands, and a fuzzy slash autocomplete.

Dynamic variables

{{date}}, {{date+3d}}, {{time}}, {{cursor}} — resolved locally, no permissions.

Smart placeholders

Text, dropdown and date fields with defaults — built visually, no syntax to memorise.

Rich text

Bold, links and lists paste formatted into rich editors and degrade to clean plain text.

Local backups

Automatic rolling snapshots with one-click restore. Import/export as JSON.

Switch in one click

Import your snippets straight from Text Blaze or Magical.

How it works

  1. 1. Save a template (with {{placeholders}} if you like).
  2. 2. Focus any text field on any site.
  3. 3. Type /shortcut, press a hotkey, or pick from the popup.
  4. 4. Fill in the blanks — done.
